# 索克家居 LOGO 资源

> **品牌**：索克家居 (i-home.life)
> **创建日期**：2026-07-08
> **设计哲学**：Architectural Stillness — 建筑静默

## 设计说明

- **核心标识**：深蓝绿色（#2D5A7B）圆角方块 + 白色「iH」字母
- **i** = innovation（创新），**H** = home（家居）
- **视觉隐喻**：i 如柱，H 如门户 — 形如建筑立面
- **圆角比例**：22.36%（iOS squircle 标准）
- **光学居中**：整体向左偏移 1.2% 补偿 H 的视觉重量
- **字体**：Instrument Sans Bold

## 使用规范

1. **最小尺寸**：印刷 24px / 屏幕 16px，低于此尺寸需使用简化版
2. **安全间距**：四周保留图标尺寸 10% 的空白
3. **背景要求**：透明背景下使用 RGBA 版本；纯色背景可使用 flat 版本
4. **禁止变形**：不得拉伸、倾斜、旋转或添加投影/发光效果
5. **禁止改色**：不得更改品牌色，深色模式使用 dark 变体
