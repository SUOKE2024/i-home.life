01信息收集

我们在您使用平台服务时收集以下类型的信息：

注册与账户信息

  • 手机号 — 用于账号注册、登录验证与身份识别。
  • 密码 — 以加密哈希形式存储，不存储明文。
  • 角色信息 — 业主、设计师、工长或管理员身份标识。

项目数据

  • 项目信息 — 项目名称、地址（支持高德地图智能补全）、户型描述等。
  • 楼层与房间数据 — 户型结构、房间名称、面积等。
  • 设计方案 — 2D/3D 设计图纸、布局方案、AI 生成的设计建议。
  • 预算与采购数据 — 预算方案、BOM 物料清单、采购订单、供应商信息。
  • 施工与结算数据 — 施工任务、日志、质检记录、结算单据、支付记录。

测量数据

  • AR 空间测量数据 — 通过 ARKit / ARCore / AR Engine 采集的房间尺寸、墙面特征、测量点坐标。
  • 扫描会话记录 — 扫描时间、设备信息、精度校验结果。
  • 手动录入数据 — 用户手动输入的房间尺寸与户型信息。

文件上传

  • 工程文件 — 图纸（DXF）、合同、质检报告、变更单等用户主动上传的文件。
  • 施工照片 — 用于 AI 图像质检的现场施工照片。
  • VR 全景图 — 用于全景渲染的等距柱状全景图。
设备信息

在 AR 测量功能中，我们会读取设备能力信息（如是否支持 AR、传感器类型），用于功能适配与降级策略判断。我们不收集设备唯一标识符用于广告追踪。

02信息使用

我们收集的信息仅用于以下目的：

  • 提供装修服务 — 创建和管理您的装修项目，执行测量、设计、预算、采购、施工与结算等业务流程。
  • AI 分析与辅助决策 — 使用 DeepSeek AI 模型分析您的设计需求、户型数据与预算信息，生成设计方案、预算建议与施工计划。AI 分析在服务端进行，您的数据不会用于训练第三方 AI 模型。
  • 项目管理与协作 — 在您授权的范围内，将项目信息共享给参与项目的施工方、供应商或设计师，支持多方协作。
  • 服务改进 — 收集使用情况统计与错误日志，用于优化平台功能与用户体验。
  • 安全防护 — 用于身份验证、防止欺诈、保障账户与交易安全。
  • 通知与沟通 — 向您发送项目进度通知、施工预警、结算提醒等服务消息。
重要说明

AI 生成的设计方案、预算建议与施工计划仅供参考，不构成专业工程建议。最终方案应由具备资质的设计师或工程师审核确认。

03信息存储与保护

我们采取行业标准的措施保护您的个人信息安全。

数据存储

  • 数据库 — 开发环境使用 SQLite，生产环境支持 PostgreSQL。数据库部署在服务器本地或私有网络中，不使用第三方托管数据库。
  • 文件存储 — 用户上传的工程文件与图片默认存储在服务器本地文件系统中。生产环境可选配置阿里云 OSS 对象存储，文件传输均经过 HTTPS 加密。
  • 缓存 — 平台可选使用 Redis 缓存加速数据访问，缓存数据不包含敏感信息，且有过期机制。未配置 Redis 时自动降级为内存缓存。
  • 数据表 — 平台使用 72 张数据表分类存储不同业务数据，实现最小化访问原则。

安全措施

  • PASETO 令牌认证 — 采用 PASETO v4 协议（local 模式）进行身份认证，令牌内容对称加密，比传统 JWT 更安全。令牌有效期为 24 小时。
  • 密码加密 — 用户密码以哈希形式存储，不存储明文密码。
  • HTTPS 传输加密 — 生产环境通过 Nginx 配置 SSL/TLS，所有数据传输均经过加密。
  • 安全响应头 — Nginx 配置 X-Frame-Options、CSP、X-Content-Type-Options 等安全头，防御 XSS、点击劫持等攻击。
  • 访问控制 — 基于角色的访问控制（RBAC），不同角色（业主/设计师/工长/管理员）拥有不同数据访问权限。
  • CORS 白名单 — 生产环境仅允许白名单域名访问 API。
数据保留

您的数据将在您使用平台服务期间持续保留。如您注销账户或请求删除数据，我们将在合理期限内删除您的个人信息。法律法规另有规定的除外。

04信息共享

我们尊重您的隐私，不会向第三方出售您的个人信息。在以下情形中，我们可能共享您的部分信息：

  • 经您授权的共享 — 当您在项目中邀请施工方、供应商或设计师协作时，平台会在您授权范围内将相关项目信息（如设计方案、预算、施工要求）共享给被邀请方。
  • 担保支付场景 — 在采购增强模块的担保支付流程中，必要的订单与支付信息会在您、供应商与平台之间共享，以完成资金托管与释放。
  • 法律要求 — 因法律法规要求、诉讼需要或行政机关强制要求时，我们可能披露必要的信息。
共享原则

所有信息共享均以最小必要为原则。施工方与供应商仅能查看与其参与项目相关的数据，无法访问您的其他项目或个人敏感信息。

05用户权利

您对自己的个人信息享有以下权利：

权利说明
查看权您可随时在个人中心查看您的账户信息、项目数据与操作记录。
修改权您可修改个人信息、项目资料与设计方案。部分关键数据（如已结算的金额）修改需遵循平台流程。
删除权您可请求删除您的账户及相关数据。已上传的工程文件、施工记录等可单独删除。
导出权您可导出项目数据（设计方案 DXF、物料清单 Excel、预算报告等）留存备份。
撤回授权您可随时撤回对施工方或供应商的项目协作授权，终止信息共享。
注销权您可注销账户。注销后您的个人信息将被删除或匿名化处理。
行使权利

如需行使上述权利，请通过本政策末尾的联系方式与我们取得联系。我们将在 15 个工作日内回复您的请求。

06Cookie 与本地存储

平台使用浏览器本地存储（localStorage）而非传统 Cookie 来管理会话状态。

本地存储内容

  • PASETO 令牌 — 登录成功后，PASETO 认证令牌存储在浏览器 localStorage 中，用于后续 API 请求的身份验证。令牌有效期为 24 小时。
  • 用户偏好 — 如界面主题、最近查看的项目等非敏感偏好设置。
  • 离线缓存 — Web 端缓存静态资源（JS/CSS）加速页面加载；Flutter App 使用 SharedPreferences 缓存项目列表等非敏感数据，在网络不可用时提供降级浏览。缓存数据不包含密码或敏感个人信息。
管理方式

您可通过浏览器设置清除 localStorage 数据。清除后需重新登录。我们不使用追踪型 Cookie，不进行跨站广告追踪。

07第三方服务

平台集成了以下第三方服务，这些服务可能收集和处理您的部分信息：

高德地图 API

  • 用途 — 地址智能补全、地理位置定位。
  • 共享信息 — 您输入的地址关键词与 IP 地址（用于定位）。
  • 隐私政策 — 受高德地图隐私政策约束。

DeepSeek AI

  • 用途 — AI 设计方案生成、预算分析、施工计划建议等智能服务。
  • 共享信息 — 您的设计需求描述、户型数据、预算参数等业务上下文（用于 AI 推理）。
  • 数据处理 — AI 推理在 DeepSeek 服务端进行，数据不用于模型训练。
  • Mock 模式 — 未配置 API Key 时，平台使用本地规则引擎替代 AI 调用，不发送任何数据到外部。
注意

第三方服务提供商有其独立的隐私政策。我们建议您查阅相关服务的隐私政策，了解其数据处理实践。我们不对第三方服务的隐私实践承担责任。
